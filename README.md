# Veatriki, a search and export tool for PHI

Quickstart:
1. Download Python 3.7+ from https://www.python.org/downloads/ and install
2. Download iphi.json from https://drive.google.com/drive/folders/1WupkpBTP7BTGTqAwKQ8BSFrCaQTbKX9c
3. Download veatriki.py
4. In your terminal, navigate to the folder that contains both files
5. Run python veatriki.py
